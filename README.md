🛠️ Especificaciones Tecnicas del ADN Camerunes
Este contrato documenta la precision quimica necesaria para neutralizar toxinas y alcaloides naturales:

Intensidad de Lavado (uint256): Registra el proceso de desamargado de hojas como el Ndole. Un nivel 8 indica un lavado profundo que preserva la estructura de la hoja pero elimina el amargor excesivo.

Usa Envoltorio de Hoja (bool): Validador del uso de hojas de platano o Marantaceae, que funcionan como recipientes biodegradables y aislantes termicos.

Fermentacion Previa (bool): Identificador critico para derivados de la yuca (como el Bobolo o Miondo), donde la actividad enzimatica es necesaria antes de la coccion.

Sistema de Feedback: Funciones de darLike y darDislike para la participacion de la comunidad.
